# 01 - Data Types

This folder contains SystemVerilog examples that demonstrate the commonly used built-in and user-defined data types. Each example focuses on a single concept and can be compiled and simulated independently.

## Files

| # | File                           | Concept Demonstrated                                           |
| - | ------------------------------  -------------------------------------------------------------- |
| 1 | `01_default_values.sv`         - Default values of SystemVerilog data types                     |
| 2 | `02_2state_vs_4state.sv`       - Comparison of 2-state (`bit`) and 4-state (`logic`) data types |
| 3 | `03_arrays_packed_unpacked.sv` - Packed and unpacked array declarations and usage               |
| 4 | `04_dynamic_array.sv`          - Dynamic array allocation, initialization, and resizing         |
| 5 | `05_queue.sv`                  - Queue declaration and common queue methods                     |
| 6 | `06_associative_array.sv`      - Associative arrays with integer and string indexing            |
| 7 | `07_struct_union.sv`           - User-defined data types using `struct` and `union`             |
| 8 | `08_enum.sv`                   - Enumerated data types and built-in enum methods                |
| 9 | `09_typedef_and_casting.sv`    - Type aliases using `typedef` and data type casting             |

## Key Concepts

* Built-in data types
* Default values
* 2-state and 4-state data types
* Packed and unpacked arrays
* Dynamic arrays
* Queues
* Associative arrays
* User-defined data types (`struct`, `union`, `enum`, `typedef`)
* Type casting
